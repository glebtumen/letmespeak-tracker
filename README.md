# Letmespeak Tracker
## Crypto and learn

Letmespeak Tracker - бот для отслеживания цены на предметы маркетплейса LetMeSpeak.

## Features
- Следит за самыми низкими ценами
- Уведомляет в сообщениях
- ✨Бонус: страшный код на 1000 строк в одном файле
